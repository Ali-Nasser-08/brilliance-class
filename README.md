# Brilliant Class

A fun English learning app with colorful animations and interactive exercises.

## Features

- **Revision Mode** - Practice vocabulary and grammar for Units 7-12
- **Breathing Exercises** - Psychological Sigh and Box Breathing to help focus
- **Idle Animations** - Beautiful typewriter stories, word flash cards, and multi-language displays
- **Dynamic Colors** - Smooth color transitions that change as you use the app

## Getting Started
